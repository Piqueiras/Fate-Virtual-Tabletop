## Fate VTT es un tablero virtual para el sistema de rol [Fate](https://fate-srd.com/fate-core) desarrollado en Rails 8.1.

Cuenta con un sistema de creación de personajes con las capacidades más básicas, además de la posibilidad de crear salas para jugar una partida.

En dichas salas se permite importar personajes y moverlos por el tablero, además de lanzar los dados, dibujar, crear objetos y gestionar los puntos de destino de los personajes.
